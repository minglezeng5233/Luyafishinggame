## 🎯 游戏特色

### 🎣 核心玩法
- **真实钓鱼体验**: 抛竿→等待→收线完整流程
- **随机鱼类系统**: 8种不同稀有度鱼类
- **动态等待时间**: 3-7秒随机等待，增加真实感
- **计时器显示**: 实时显示钓鱼时间

### 🏆 成长系统
- **等级系统**: 经验值升级，自动计算
- **金币经济**: 出售鱼类获得金币
- **升级奖励**: 每次升级获得金币奖励
- **进度显示**: 实时经验条和进度

### 🐟 鱼类图鉴
- **8种鱼类**: 鲤鱼、黑鲈、鳜鱼、鲶鱼、海鲈、金枪鱼、鲨鱼、龙鱼
- **4个稀有度**: 普通、稀有、珍稀、传说
- **收集进度**: 实时显示收集完成度
- **详细信息**: 每种鱼的重量和价值记录

### 🛒 钓具商店
- **基础钓具**: 入门钓竿和拟饵
- **金币购买**: 使用游戏内金币购买
- **扩展接口**: 预留更多商品添加

## 🎮 操作说明

### 🎣 钓鱼操作
1. 点击"开始钓鱼"进入钓鱼界面
2. 点击大型钓鱼按钮开始抛竿
3. 等待系统提示"有动静了"
4. 自动收线，显示捕获结果
5. 查看获得金币和经验

### 📚 图鉴查看
1. 在主界面点击"图鉴"
2. 查看已收集和未收集的鱼类
3. 绿色背景表示已收集
4. 灰色背景表示未收集

### 🛒 商店购买
1. 在主界面点击"商店"
2. 查看可用钓具
3. 确认金币充足后点击购买
